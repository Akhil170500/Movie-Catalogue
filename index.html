This Site

This is an movie catalogue website where you can get information about particular movie. Bootstrap and OMDB API is use to build this web page.

OMDB

The Open Movie Database. The OMDb API is a free web service to obtain movie information, all content and images on the site are contributed and maintained by our users.

Purpose

Is to create a simple web page to fetch information about any movie using an API which is easy to use for a new user.
